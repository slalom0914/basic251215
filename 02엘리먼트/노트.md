## 엘리먼트
https://ko.reactjs.org/docs/fragments.html#gatsby-focus-wrapper

React에서 컴포넌트가 여러 엘리먼트를 반환하는 것은 흔한 패턴입니다. <br>
Fragments는 DOM에 별도의 노드를 추가하지 않고 여러 자식을 그룹화할 수 있습니다.<br>
```js
render() {
  return (
    <React.Fragment>
      <ChildA />
      <ChildB />
      <ChildC />
    </React.Fragment>
  );
}
```
이를 선언하는 새로운 단축 문법이 있습니다.
```js
render() {
  return (
    <>
      <ChildA />
      <ChildB />
      <ChildC />
    </>
  );
}
```
